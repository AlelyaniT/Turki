    Current research projects

    1.Crowdsourcing Task Quality

    In this project, I apply Natural Langauge Processing techniques to understand the quality of the task design. Prediction models can be proposed to test diffrent features. For instance, how contextual features can impact the task success or fail. The structure of a task is another factor I consider. How cognitive load theory can be applied on this context?.

    2.Task Pricing on Crowdsourcing

    One of the challenges on crowdsourcing is what pricing strategies we should apply. I takle this problem by proposing a new techniques which I call "Requirements-based models techniques". I argue that by having the requirements only, we can come up with a reasonable pricing technique. This work will overcome previous study limitation.

    The following machine learning algorithms are applied on topic modeling analysis to predict the software crowdsourcing pricing.

    • Logistic Regression (LR)
    • Classification and Regression Trees (CART)
    • Gaussian Naive Bayes (NB)
    • Support Vector Machines (SVM)
    • K-Nearest Neighbors (KNN)
    • Linear Discriminant Analysis (LDA)
    • Neural Network (NN)

      • 3.Who are the crowd and what motivate them

      This project aims to understand crowd motivation in the context of software crowdsourcing where complex software projects can be developed. The findings of this study will guide future research into a better understanding of the crowd and how we can improve design strategies.
